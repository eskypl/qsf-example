QSF examples

A guide designed for website owners who became eSky.pl affiliates and want to implement eSky's QSF (Quick Search Form). This guide will introduce to you basic concepts required to correctly implement QSF.

General information

This page contains various examples of QSFs used by eSky.pl services. You can explore those examples in many ways, from just browsing to cloning or forking repository. You can also use them on your own webpage, however remember that code in this repository is provided as is, without any warranty or support.

Guide and examples are focused only on technical aspects of building QSF. Any design & usability topics are not covered here.

Flights

  • Documentation - basic knowledge how to assemble QSF for flights service.
  • Basic example - contains only valid HTML markup with JavaScript required for interaction with Trip type controls.
